# 🎨 Иконки для социальных сетей

## 📁 Расположение

Иконки хранятся в папке:
```
storage/app/public/icons/
```

Публичный доступ через symlink:
```
public/storage/icons/
```

## 📦 Текущие иконки

В папке уже есть следующие иконки:
- `MAX.png` — MAX
- `OK.png` — Одноклассники
- `RUTUBE.png` — Rutube
- `VK.png` — ВКонтакте

## 🚀 Как добавить новую иконку

### Способ 1: Через админ-панель Filament

1. Откройте мероприятие в режиме редактирования
2. Перейдите в раздел **"Социальные сети"**
3. Нажмите кнопку **"Загрузить иконку"** в заголовке секции
4. Выберите файл изображения (PNG, JPG, SVG, WEBP)
5. Иконка будет сохранена в `storage/app/public/icons/`

### Способ 2: Вручную через файловую систему

1. Скопируйте файл иконки в `storage/app/public/icons/`
2. Убедитесь, что файл имеет одно из расширений: `.png`, `.jpg`, `.jpeg`, `.svg`, `.webp`, `.gif`
3. Имя файла будет использовано как название иконки (без расширения)

**Пример:**
```bash
cp telegram-icon.png storage/app/public/icons/
```

Теперь иконка "Telegram icon" будет доступна в выпадающем списке.

## 🎯 Использование в форме

В форме редактирования мероприятия:

1. Раздел **"Социальные сети"** → кнопка **"Добавить соцсеть"**
2. Заполните поля:
   - **Платформа** — выберите из списка (Facebook, Instagram, VK и т.д.)
   - **URL страницы** — ссылка на социальную сеть
   - **Иконка** — выберите из доступных иконок в папке `storage/app/public/icons`

## 🔄 Обновление списка иконок

Список иконок загружается динамически при загрузке формы. Если вы добавили иконку вручную, просто обновите страницу браузера.

## 🗑 Удаление иконки

Для удаления иконки:
```bash
rm storage/app/public/icons/имя-иконки.png
```

**Важно:** Удалите все версии иконки с разными расширениями, если они есть.

## 📝 Рекомендации по именам

Используйте понятные имена для иконок:
- `facebook.png`
- `instagram.png`
- `vk.png`
- `telegram.png`
- `youtube.png`
- `custom-promo.png`

Имена будут отображаться в выпадающем списке с заглавной буквы и пробелами вместо дефисов.

## 🎨 Рекомендации по размеру

- Оптимальный размер: 64x64px или 128x128px
- Формат: PNG с прозрачностью или SVG
- Максимальный размер файла: 50KB

## ⚠️ Важные замечания

1. Иконки автоматически доступны для всех мероприятий
2. Если иконка не отображается в списке:
   - Проверьте права доступа к файлу
   - Убедитесь, что расширение файла поддерживается
   - Очистите кэш: `php artisan cache:clear`
