# План: Рекомендации 4, 5, 6 для главной страницы

## Задачи

### 4. Lazy-load Plyr (~20KB)
Plyr импортируется безусловно в `app.js:5-6`, хотя видео рендерится только при наличии `video_url` (`home.blade.php:114`).

**Решение:** Динамический импорт Plyr через `import()` только когда видео есть.

**Файлы:**
- `resources/js/app.js` — убрать статический `import Plyr`, добавить динамический импорт в `window.Plyr`
- `resources/views/home.blade.php:115` — заменить `new Plyr($refs.player)` на вызов через `window.Plyr` (который загрузится динамически)

### 5. Замена inline style на Tailwind
Найдено 2 места с inline стилями:

| Строка | Было | Стало |
|--------|------|-------|
| `home.blade.php:255` | `style="background-color: #e5e5e5;"` | Убрать (уже есть `bg-[var(--color-background)]`) |
| `home.blade.php:226` | `style="display: none;"` | Оставить (нужен для Alpine `x-cloak` инициализации) |

**Файл:** `resources/views/home.blade.php`

### 6. CLS: добавить `decoding="async"` + `width`/`height` в галерею
Текущий тег `<img>` в галерее (`home.blade.php:202-208`) не имеет `width`, `height`, `decoding`. Это вызывает Cumulative Layout Shift.

**Решение:** Добавить `decoding="async"` и фиксированные пропорции через `aspect-ratio` CSS (реальные width/height изображений недоступны на сервере, но `aspect-ratio` + `object-cover` предотвращают CLS).

**Файл:** `resources/views/home.blade.php`

## Изменения

### `resources/js/app.js`
```diff
- import Plyr from 'plyr';
- import 'plyr/dist/plyr.css';

// ... в конце файла или в initSwiper:
+ window.Plyr = null;
+ window.loadPlyr = async function() {
+     if (!window.Plyr) {
+         const [{ default: PlyrClass }] = await Promise.all([
+             import('plyr'),
+             import('plyr/dist/plyr.css'),
+         ]);
+         window.Plyr = PlyrClass;
+     }
+     return window.Plyr;
+ };
```

### `resources/views/home.blade.php`
```diff
# Plyr lazy-load (строка 115):
- <div x-init="new Plyr($refs.player)" class="bg-[var(--color-text)]">
+ <div x-init="window.loadPlyr().then(P => new P($refs.player))" class="bg-[var(--color-text)]">

# Venue section (строка 255):
- <section id="venue" class="bg-[var(--color-background)] pt-20 pb-20 text-[var(--color-text)]" style="background-color: #e5e5e5;">
+ <section id="venue" class="bg-[#e5e5e5] pt-20 pb-20 text-[var(--color-text)]">

# Gallery images (строка 202-208):
  <img
      src="{{ $image }}"
      alt="{{ $activeEvent->title }}"
-     class="w-full h-auto object-cover shadow-md gallery-image cursor-pointer brightness-90 hover:brightness-100 transition-all"
-     loading="lazy"
+     class="w-full aspect-square object-cover shadow-md gallery-image cursor-pointer brightness-90 hover:brightness-100 transition-all"
+     loading="lazy"
+     decoding="async"
      @click="index = {{ $idx }}; open = true"
  />
```
